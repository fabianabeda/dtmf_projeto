Introdução

O presente projeto técnico detalha o desenvolvimento de um sistema de detecção de tons Dual-Tone Multi-Frequency (DTMF) utilizando técnicas avançadas de processamento digital de sinais [1]. A discagem por tom, amplamente empregada em sistemas de telefonia e comunicação, baseia-se na transmissão de pares de frequências para representar dígitos e comandos [2].

A capacidade de decodificar esses tons de forma precisa e robusta é fundamental para a funcionalidade de diversas aplicações, desde sistemas de resposta de voz interativa (IVR) até controle remoto via áudio [3]. Este projeto demonstra a implementação computacional de um detector DTMF em Python, focando na aplicação da Transformada Rápida de Fourier (FFT) para análise espectral [4] e filtros de Butterworth para separação de frequências [5].

Objetivos do Projeto

  • Implementar um detector DTMF robusto e preciso
  • Aplicar conceitos de processamento digital de sinais
  • Demonstrar a eficácia da FFT na análise espectral
  • Validar a performance em ambientes ruidosos

Fundamentação Teórica

Sistema DTMF

O sistema DTMF utiliza dois grupos de frequências: baixa (697-941 Hz) e alta (1209-1477 Hz) [1]. Cada tecla gera uma combinação única de duas frequências simultâneas, permitindo identificação precisa mesmo na presença de ruído [2].

Transformada de Fourier

A Transformada Rápida de Fourier (FFT) é fundamental para converter sinais do domínio do tempo para o domínio da frequência, revelando as componentes espectrais presentes no sinal DTMF e permitindo a identificação dos tons [4].

FFT: X(k) = Σ x(n) · e^(-j2πkn/N)

onde N é o número de amostras e k representa o índice de frequência [6].

Filtros de Butterworth

Os filtros de Butterworth oferecem resposta em frequência maximamente plana na banda passante, sendo ideais para separar os grupos de baixa e alta frequência do sinal DTMF sem introduzir distorções significativas [5].

Passa-Baixa: Isola frequências < 1000 Hz
Passa-Alta: Isola frequências > 1100 Hz

Geração de Sinais DTMF

Implementação da função para gerar tons DTMF puros através da soma de duas senoides correspondentes às frequências de linha e coluna de cada dígito.


import numpy as np

# Frequências DTMF padrão
FREQUENCIAS_DTMF = {
    '1': (697, 1209), '2': (697, 1336), '3': (697, 1477),
    '4': (770, 1209), '5': (770, 1336), '6': (770, 1477),
    '7': (852, 1209), '8': (852, 1336), '9': (852, 1477),
    '*': (941, 1209), '0': (941, 1336), '#': (941, 1477),
}

def gerar_tom_dtmf(digito, duracao, taxa_amostragem):
    if digito not in FREQUENCIAS_DTMF:
        raise ValueError(f"Dígito inválido: {digito}")
    
    f_baixa, f_alta = FREQUENCIAS_DTMF[digito]
    
    t = np.linspace(0, duracao, int(taxa_amostragem * duracao), endpoint=False)
    return (np.sin(2 * np.pi * f_baixa * t) + np.sin(2 * np.pi * f_alta * t)) / 2

Filtragem Digital

Aplicação de filtros de Butterworth para separar os grupos de baixa e alta frequência, preparando o sinal para análise espectral precisa.


import numpy as np # Importado para o escopo do snippet
from scipy import signal

def aplicar_passabaixa_butter(sinal, freq_corte, taxa_amostragem, ordem=5):
    nyquist = taxa_amostragem / 2
    freq_norm = freq_corte / nyquist
    freq_norm = min(freq_norm, 0.99) # Garante que a frequência normalizada está no intervalo válido
    b, a = signal.butter(ordem, freq_norm, btype='low')
    return signal.filtfilt(b, a, sinal)

def aplicar_passalta_butter(sinal, freq_corte, taxa_amostragem, ordem=5):
    nyquist = taxa_amostragem / 2
    freq_norm = freq_corte / nyquist
    freq_norm = min(freq_norm, 0.99) # Garante que a frequência normalizada está no intervalo válido
    b, a = signal.butter(ordem, freq_norm, btype='high')
    return signal.filtfilt(b, a, sinal)

def aplicar_passafaixa_butter(sinal, freq_baixa, freq_alta, taxa_amostragem, ordem=5):
    nyquist = taxa_amostragem / 2
    freq_baixa_norm = freq_baixa / nyquist
    freq_alta_norm = freq_alta / nyquist
    freq_baixa_norm = max(freq_baixa_norm, 0.01)
    freq_alta_norm = min(freq_alta_norm, 0.99)
    b, a = signal.butter(ordem, [freq_baixa_norm, freq_alta_norm], btype='band')
    return signal.filtfilt(b, a, sinal)

Análise Espectral

Utilização da FFT para identificar picos de frequência dominantes em cada grupo, permitindo a decodificação do dígito DTMF.


import numpy as np
from scipy.fft import fft

def detectar_pico_freq(sinal, taxa_amostragem, faixa):
    N = len(sinal)
    freq = np.fft.fftfreq(N, 1 / taxa_amostragem)
    espectro = fft(sinal)
    # Filtra por frequências positivas e dentro da faixa de interesse
    indices = np.where((freq >= faixa[0]) & (freq <= faixa[1])) 
    freq_pos = freq[indices]
    amp = 2.0 / N * np.abs(espectro[indices])
    
    if len(freq_pos) > 0: # Usar freq_pos, que já está filtrada pela faixa
        idx_pico = np.argmax(amp) # max na amplitude já filtrada
        pico_amplitude = amp[idx_pico]
        
        # Limiar de amplitude (ajuste este valor conforme sua análise e testes)
        limiar_amplitude = 0.15 # Exemplo de limiar, pode precisar de ajuste
        
        if pico_amplitude < limiar_amplitude:
            return None # Sinal muito fraco, ou ruído, não detecta a frequência
        
        return freq_pos[idx_pico]
    
    return None # Nenhuma frequência válida na faixa

Validação com Ruído

Testes de robustez com adição de ruído branco em diferentes níveis de SNR para simular condições reais de operação.


import numpy as np

def adicionar_ruido_branco(sinal, snr_db):
    potencia_sinal = np.mean(sinal**2)
    snr_linear = 10 ** (snr_db / 10) # Converte dB para linear
    potencia_ruido = potencia_sinal / snr_linear
    # Gera ruído gaussiano com a potência calculada
    ruido = np.random.normal(0, np.sqrt(potencia_ruido), len(sinal))
    return sinal + ruido

Filtragem com Filtro RC e Convolução (Duração do Sinal: 0.5s)

Além dos filtros digitais de Butterworth, foi implementado um filtro analógico passa-baixa do tipo RC, com resistência de 600 ohms e frequência de corte de aproximadamente 1000 Hz, conforme os padrões da telefonia. A resposta ao impulso do filtro foi amostrada e usada em uma operação de convolução no domínio do tempo para simular o comportamento físico do circuito.

A equação da resposta ao impulso utilizada foi:

h(t) = (1/RC) · e-t/RC · u(t)

Essa função foi discretizada e convoluída com o sinal DTMF, gerando um novo sinal filtrado que preserva apenas a frequência de baixa frequência (até 1 kHz). A convolução foi realizada com:

sinal_filtrado = np.convolve(sinal, h, mode='same')
Figura 5: Comparação do sinal original e do sinal filtrado com convolução por filtro RC.

O resultado mostra atenuação das componentes de alta frequência, confirmando a efetividade da filtragem analógica simulada. Esse experimento atende ao Exercício 1 proposto no projeto.

Conclusão

O projeto demonstrou com sucesso a implementação de um detector de tons DTMF robusto e preciso utilizando conceitos fundamentais de processamento digital de sinais. A combinação da Transformada Rápida de Fourier (FFT) para análise espectral e dos filtros de Butterworth para separação de frequências provou ser uma abordagem eficaz.

A acurácia de 100% alcançada em todos os testes, tanto com sinais puros quanto com sinais ruidosos em diferentes níveis de SNR, valida a metodologia empregada e demonstra a robustez do sistema implementado.
