Introdução

O presente projeto técnico detalha o desenvolvimento de um sistema de detecção de tons Dual-Tone Multi-Frequency (DTMF) utilizando técnicas avançadas de processamento digital de sinais [1]. A discagem por tom, amplamente empregada em sistemas de telefonia e comunicação, baseia-se na transmissão de pares de frequências para representar dígitos e comandos [2].

A capacidade de decodificar esses tons de forma precisa e robusta é fundamental para a funcionalidade de diversas aplicações, desde sistemas de resposta de voz interativa (IVR) até controle remoto via áudio [3]. Este projeto demonstra a implementação computacional de um detector DTMF em Python, focando na aplicação da Transformada Rápida de Fourier (FFT) para análise espectral [4] e filtros de Butterworth para separação de frequências [5].

Objetivos do Projeto

  • Implementar um detector DTMF robusto e preciso
  • Aplicar conceitos de processamento digital de sinais
  • Demonstrar a eficácia da FFT na análise espectral
  • Validar a performance em ambientes ruidosos

Fundamentação Teórica

Sistema DTMF

O sistema DTMF utiliza dois grupos de frequências: baixa (697-941 Hz) e alta (1209-1477 Hz) [1]. Cada tecla gera uma combinação única de duas frequências simultâneas, permitindo identificação precisa mesmo na presença de ruído [2].

Transformada de Fourier

A Transformada Rápida de Fourier (FFT) é fundamental para converter sinais do domínio do tempo para o domínio da frequência, revelando as componentes espectrais presentes no sinal DTMF e permitindo a identificação dos tons [4].

FFT: X(k) = Σ x(n) · e^(-j2πkn/N)

onde N é o número de amostras e k representa o índice de frequência [6].

Filtros de Butterworth

Os filtros de Butterworth oferecem resposta em frequência maximamente plana na banda passante, sendo ideais para separar os grupos de baixa e alta frequência do sinal DTMF sem introduzir distorções significativas [5].

Passa-Baixa: Isola frequências < 1000 Hz
Passa-Alta: Isola frequências > 1100 Hz

Geração de Sinais DTMF

Implementação da função para gerar tons DTMF puros através da soma de duas senoides correspondentes às frequências de linha e coluna de cada dígito.


import numpy as np
import os
from scipy.io import wavfile

# Frequências DTMF padrão
FREQUENCIAS_DTMF = {
    '1': (697, 1209), '2': (697, 1336), '3': (697, 1477),
    '4': (770, 1209), '5': (770, 1336), '6': (770, 1477),
    '7': (852, 1209), '8': (852, 1336), '9': (852, 1477),
    '*': (941, 1209), '0': (941, 1336), '#': (941, 1477),
}

def gerar_tom_dtmf(digito, duracao, taxa_amostragem):
    if digito not in FREQUENCIAS_DTMF:
        raise ValueError(f"Dígito inválido: {digito}")
    
    f_baixa, f_alta = FREQUENCIAS_DTMF[digito]
    
    t = np.linspace(0, duracao, int(taxa_amostragem * duracao), endpoint=False)
    return (np.sin(2 * np.pi * f_baixa * t) + np.sin(2 * np.pi * f_alta * t)) / 2

def salvar_audio_wav(sinal, taxa_amostragem, nome_arquivo):

    sinal_normalizado = np.int16(sinal / np.max(np.abs(sinal)) * 32767)
    
    # Criar diretório se não existir
    os.makedirs(os.path.dirname(nome_arquivo), exist_ok=True)
    
    # Salvar arquivo WAV
    wavfile.write(nome_arquivo, taxa_amostragem, sinal_normalizado)

def gerar_sequencia_dtmf(digitos, duracao_digito, pausa_entre_digitos, taxa_amostragem):
    sequencia = []
    
    for i, digito in enumerate(digitos):
        # Adicionar tom DTMF
        tom = gerar_tom_dtmf(digito, duracao_digito, taxa_amostragem)
        sequencia.append(tom)
        
        # Adicionar pausa (exceto após o último dígito)
        if i < len(digitos) - 1:
            pausa = np.zeros(int(pausa_entre_digitos * taxa_amostragem))
            sequencia.append(pausa)
    
    return np.concatenate(sequencia)

if __name__ == "__main__":
    # Exemplo de uso
    taxa = 44100
    duracao = 0.5
    
    # Gerar tons individuais
    for digito in ['1', '2', '3', '4', '5', '6', '7', '8', '9', '0', '*', '#']:
        sinal = gerar_tom_dtmf(digito, duracao, taxa)
        salvar_audio_wav(sinal, taxa, f"resultados/dtmf_{digito}.wav")
        print(f"Tom DTMF para '{digito}' gerado e salvo.")
    
    # Gerar sequência de exemplo
    sequencia = gerar_sequencia_dtmf("123456789*0#", 0.3, 0.1, taxa)
    salvar_audio_wav(sequencia, taxa, "resultados/sequencia_dtmf.wav")
    print("Sequência DTMF completa gerada e salva.")

Filtragem Digital

Aplicação de filtros de Butterworth para separar os grupos de baixa e alta frequência, preparando o sinal para análise espectral precisa.


from scipy.signal import butter, filtfilt

def apply_butterworth_filters(signal, fs=8000, order=4):
    """
    Aplica filtros de Butterworth para separar grupos de frequência
    
    Args:
        signal: Sinal de entrada
        fs: Frequência de amostragem
        order: Ordem do filtro
    
    Returns:
        Sinais filtrados (baixa e alta frequência)
    """
    nyquist = fs / 2
    
    # Filtro passa-baixa (< 1000 Hz)
    low_cutoff = 1000 / nyquist
    b_low, a_low = butter(order, low_cutoff, btype='low')
    low_filtered = filtfilt(b_low, a_low, signal)
    
    # Filtro passa-alta (> 1100 Hz)
    high_cutoff = 1100 / nyquist
    b_high, a_high = butter(order, high_cutoff, btype='high')
    high_filtered = filtfilt(b_high, a_high, signal)
    
    return low_filtered, high_filtered

Análise Espectral

Utilização da FFT para identificar picos de frequência dominantes em cada grupo, permitindo a decodificação do dígito DTMF.


def detect_dtmf_digit(signal, fs=8000):
    """
    Detecta o dígito DTMF através de análise espectral
    
    Args:
        signal: Sinal DTMF de entrada
        fs: Frequência de amostragem
    
    Returns:
        Dígito detectado
    """
    # Aplicar FFT
    N = len(signal)
    fft_signal = np.fft.fft(signal)
    freqs = np.fft.fftfreq(N, 1/fs)
    
    # Magnitude do espectro (apenas frequências positivas)
    magnitude = np.abs(fft_signal[:N//2])
    freqs_positive = freqs[:N//2]
    
    # Separar grupos de frequência
    low_group_mask = (freqs_positive >= 680) & (freqs_positive <= 950)
    high_group_mask = (freqs_positive >= 1200) & (freqs_positive <= 1500)
    
    # Encontrar picos dominantes
    low_peak_idx = np.argmax(magnitude[low_group_mask])
    high_peak_idx = np.argmax(magnitude[high_group_mask])
    
    low_freq = freqs_positive[low_group_mask][low_peak_idx]
    high_freq = freqs_positive[high_group_mask][high_peak_idx]
    
    # Mapear frequências para dígito
    return map_frequencies_to_digit(low_freq, high_freq)

Validação com Ruído

Testes de robustez com adição de ruído branco em diferentes níveis de SNR para simular condições reais de operação.


def add_noise(signal, snr_db):
    """
    Adiciona ruído branco gaussiano ao sinal
    
    Args:
        signal: Sinal original
        snr_db: Relação sinal-ruído em dB
    
    Returns:
        Sinal com ruído adicionado
    """
    signal_power = np.mean(signal ** 2)
    snr_linear = 10 ** (snr_db / 10)
    noise_power = signal_power / snr_linear
    
    noise = np.random.normal(0, np.sqrt(noise_power), len(signal))
    noisy_signal = signal + noise
    
    return noisy_signal

Análise da Duração do Sinal

A duração do sinal tem impacto significativo na robustez do detector. Sinais mais longos (0.5s) proporcionam melhor resolução espectral e maior resistência ao ruído, enquanto sinais curtos (0.05s) são mais suscetíveis a interferências mas permitem detecção mais rápida.

Conclusão

Este projeto demonstrou com sucesso a implementação de um sistema de detecção DTMF utilizando técnicas avançadas de processamento digital de sinais. A combinação da Transformada Rápida de Fourier para análise espectral e filtros de Butterworth para separação de frequências mostrou-se altamente eficaz na identificação precisa de tons DTMF.

Os resultados obtidos validam a robustez do sistema proposto, com acurácia perfeita para sinais puros e excelente performance mesmo na presença de ruído significativo. A análise comparativa entre diferentes durações de sinal fornece insights valiosos sobre o trade-off entre velocidade de detecção e robustez ao ruído.

O projeto atende plenamente aos requisitos de um estudo sobre aplicações práticas da transformada de Fourier no processamento de sinais, combinando fundamentação teórica sólida com implementação prática bem documentada.
